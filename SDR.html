LPU5 SDR Range Finder – This module uses RTL-SDR hardware via COM ports for frequency monitoring and signal strength analysis. Through cross-bearing with multiple devices (via MQTT), the approximate position of a signal source can be triangulated.
Requirements: RTL-SDR Stick, Raspberry Pi, MQTT Broker (Mosquitto).
RTL-SDR driver not installed — measurements not possible

Install one of the following options and restart the server:
  1. Python library (recommended):
    pip install pyrtlsdr numpy
  2. rtl-sdr CLI tools (Linux/Raspberry Pi):
    sudo apt install rtl-sdr
  3. Windows: Download RTL-SDR Blog Windows binaries, place rtl_tcp.exe + rtlsdr.dll next to api.py, and install the WinUSB driver with Zadig. The server will auto-start rtl_tcp on next connect.
  4. rtl_tcp (manual / remote device):
    rtl_tcp -a 0.0.0.0
    Then set the rtl_tcp Host field below to the device IP and click Connect.

Windows 10/11 – RTL-SDR Setup-Anleitung

1. SDR-Software & Binaries vorbereiten

  • Download: Laden Sie die RTL-SDR Blog Windows Binaries (Release.zip) herunter.
  • Entpacken: Öffnen Sie die ZIP-Datei und navigieren Sie in den Ordner x64.
  • Speichern: Kopieren Sie alle Dateien aus dem x64-Ordner (rtl_tcp.exe, rtlsdr.dll, libusb-1.0.dll) direkt in Ihr Projektverzeichnis, z. B.:
    C:\Users\<IhrName>\Desktop\LPU_5_Systems\lpu5-tactical

2. Zadig: USB-Treiber flashen (Wichtigster Schritt)

Windows erkennt den RTL-SDR-Stick oft als TV-Empfänger. Für SDR-Zwecke muss der Treiber auf WinUSB geändert werden:

  1. Zadig starten: Zadig.exe öffnen.
  2. Geräte auflisten: Menü Options → List All Devices aktivieren.
  3. Gerät wählen: Im Dropdown Bulk-In, Interface (Interface 0) auswählen.
  4. Treiber setzen: Im rechten Dropdown-Feld (neben dem Pfeil-Symbol →) WinUSB auswählen.
  5. Flashen: Auf Replace Driver (oder Reinstall Driver) klicken und den Vorgang bestätigen.

3. Server starten & Testen

Öffnen Sie die PowerShell im Projektordner und starten Sie den TCP-Server, um die Hardware für Programme (oder Python) freizugeben: 

cd C:\Users\<IhrName>\Desktop\LPU_5_Systems\lpu5-tactical
.\rtl_tcp.exe -a 0.0.0.0

Tragen Sie anschließend im Feld rtl_tcp Host oben 127.0.0.1 (oder die IP des Geräts) ein und klicken Sie auf Connect.

Ergebnis: Wenn in der PowerShell listening... erscheint, ist der Stick korrekt geflasht und bereit für Messungen.
